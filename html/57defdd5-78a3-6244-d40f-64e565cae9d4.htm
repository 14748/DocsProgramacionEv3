      Usuarios Methods

      Methods

      AgregarPuntuacion Método para agregar puntuación a un usuario. Incrementa el contador de partidas jugadas y, en caso de victoria, el contador de partidas ganadas y la racha actual.
      AnadirUsuario Método para agregar un nuevo usuario. Verifica los requisitos y agrega el usuario a la lista interna.
      BuscarUsuario Método para buscar un usuario en la lista interna.
      EsUsuarioValido Método para verificar si un usuario es válido. Verifica si el usuario ya existe en la lista interna.
      GetRanking Método para obtener el ranking de usuarios. Ordena los usuarios según sus partidas ganadas y crea una lista de ranking.
      GuardarUsuarios Método para guardar los usuarios en el archivo.
      ValidarUsuario Método para validar un usuario existente. Verifica los datos de usuario y contraseña y realiza la validación.
